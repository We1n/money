# Budget Tracker

Веб-приложение для учёта личного бюджета.

## Возможности

- Добавление доходов и расходов
- Отслеживание баланса
- История операций с фильтрацией
- Аналитика с графиками
- Локальное хранение данных (localStorage)

## Технологии

- React + TypeScript
- Vite
- React Router
- Zustand
- Recharts

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```
