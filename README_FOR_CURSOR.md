# Budget Tracker — README FOR CURSOR

## Назначение проекта
Веб-приложение для учёта личного бюджета.
Пользователь вручную вносит доходы и расходы.
Приложение визуализирует данные и помогает анализировать траты.

ИИ не используется.
Все расчёты — по заданным правилам.

## Платформа
- Web (desktop + mobile)
- Онлайн
- Возможность PWA в будущем

## Цель MVP
Дать пользователю быстрый и наглядный ответ:
— куда уходят деньги  
— сколько осталось  
— на что тратится больше всего  

## Источник требований
Документы в папке /docs.
Каждый документ отвечает за свой слой системы.

## Правило работы
1. Сначала читать документы по порядку.
2. Реализовывать строго по описанию.
3. Не додумывать логику вне описанных требований.
