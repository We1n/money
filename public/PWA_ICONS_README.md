# PWA Icons Setup

Для полноценной работы PWA необходимо добавить иконки приложения.

## Требуемые размеры

1. **icon-192.png** - 192x192 пикселей
2. **icon-512.png** - 512x512 пикселей

## Где разместить

Поместите файлы в папку `public/`:
```
public/
  ├── icon-192.png
  ├── icon-512.png
  └── manifest.json (уже создан)
```

## Генерация иконок

Вы можете использовать:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Любой графический редактор (Photoshop, Figma, GIMP)

## Рекомендации

- Используйте простой, узнаваемый дизайн
- Фон должен быть однотонным или прозрачным
- Избегайте мелких деталей
- Иконка должна хорошо читаться в маленьком размере

После добавления иконок приложение можно будет установить на устройство как нативное приложение.
