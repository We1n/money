# Product Vision

Проблема:
Людям сложно контролировать личные финансы без перегруженных сервисов.

Решение:
Минималистичное приложение учёта бюджета без лишней логики.

Ценность:
- Быстрое внесение данных
- Наглядная аналитика
- Эстетичный интерфейс

Целевая аудитория:
Один пользователь. Личный бюджет.
